# UI Improvements Changelog

## Что изменено

### 📦 База данных

✅ **Убрано создание 4 пустых товаров**

- При каждом запуске бота sequence сбрасывается автоматически

### 🎨 Карточки товаров

✅ **Минималистичный дизайн**
- Только название товара (без "Товар 1", "Новый товар")

✅ **Тумблер в правом верхнем углу**
- Зелёный = активен

✅ **Кнопки редактирования и удаления**
- Расположены внизу карточки

### 📝 Модальное окно редактирования

✅ **Полноэкранное модальное окно**
- Открывается снизу вверх (как в iOS)
- Удобно на любых устройствах

✅ **Стрелка назад в верхнем левом углу**
- Подтверждение при выходе с несохранёнными изменениями
- Только если пользователь что-то заполнил

✅ **Кнопка Сохранить внизу**
- Фиксирована внизу
- Всегда доступна

✅ **Кешбек в процентах**
- Input с суффиксом "%"
- Валидация 0-100%

✅ **Валидация полей**
- Название обязательно
- Кешбек 0-100%
- Инструкция обязательна

## 🚀 Как запустить

```bash
# 1. Переключиться на ветку
git checkout feature/improvements
git pull origin feature/improvements

# 2. Перезапустить Docker
sudo docker-compose down -v  # -v удалит старую БД с 4 товарами
sudo docker-compose up -d --build

# 3. Проверить
sudo docker-compose logs -f bot
```
