# 🚀 Гайд по миграции на новую версию

## Что нового?

### 🎉 Основные изменения

1. **🔄 Пошаговая воронка**
   - Пользователь проходит 5 шагов по очереди
   - Каждый шаг - отдельное сообщение с кнопкой
   - Не нужно читать длинные инструкции сразу

2. **🌐 Telegram Web App (Веб-панель)**
   - Админ может настроить всё через удобный интерфейс
   - Редактирование товаров
   - Редактирование **всех** текстов бота
   - Открывается прямо в Telegram

3. **🎯 Упрощенные товары**
   - Только название + инструкция
   - Всё (ссылка, сумма, условия) в одном тексте

4. **✅ Модерация с подтверждением**
   - Админ подтверждает/отклоняет заказ
   - При одобрении - отправляет скрин перевода
   - При отклонении - указывает причину
   - **Только после подтверждения** данные идут в Google Sheets

5. **📊 Реальтайм аналитика**
   - Лист2 обновляется мгновенно (при нажатии кнопок)
   - Лист1 обновляется только при завершении заказа

## 🛠 Инструкция по миграции

### Шаг 1: Сделайте бэкап

```bash
# Бэкап тома PostgreSQL
sudo docker run --rm -v telegram-buyout-bot_postgres_data:/data -v $(pwd):/backup \
  alpine tar czf /backup/postgres_backup_$(date +%Y%m%d).tar.gz /data
```

### Шаг 2: Обновите код

```bash
cd ~/Documents/Python/telegram-buyout-bot
git fetch origin
git checkout feature/web-admin-major-refactor
```

### Шаг 3: Добавьте в .env

Добавьте новую переменную:

```env
# URL для Telegram Web App (используйте ваш домен или ngrok)
WEBAPP_URL=https://your-domain.com
```

### Шаг 4: Очистите старые данные

⚠️ **ВНИМАНИЕ**: Это удалит все товары и заказы!

```bash
sudo docker-compose down
sudo docker volume rm telegram-buyout-bot_postgres_data
```

### Шаг 5: Запустите бота

```bash
sudo docker-compose build --no-cache
sudo docker-compose up -d
sudo docker-compose logs -f bot
```

### Шаг 6: Инициализируйте конфиг

```bash
sudo docker exec -it telegram-buyout-bot-bot-1 python -m bot.utils.init_bot_config
```

### Шаг 7: Настройте товары

1. Отправьте в бот: `/admin`
2. Нажмите "📝 Открыть настройки"
3. Заполните товары (максимум 4)
4. При необходимости отредактируйте тексты

## ❓ FAQ

**Q: Как посмотреть веб-панель?**  
A: Отправьте `/admin` в бот и нажмите кнопку "📝 Открыть настройки".

**Q: Что делать, если веб-панель не открывается?**  
A: Убедитесь, что `WEBAPP_URL` в `.env` указывает на доступный HTTPS-адрес.

**Q: Как изменить текст шагов?**  
A: Откройте веб-панель, вкладка "📝 Тексты", отредактируйте нужный текст.

**Q: Как добавить 5-й товар?**  
A: По ТЗ только 4 товара. Если нужно больше - напишите разработчику.
